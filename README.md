# BscSwap 工具说明  
一个用于 Binance Smart Chain (BSC) 的自动化交易工具，支持原生交易发送、合约调用、Swap 交易执行等功能，适用于 Web3 开发、自动化交易、MEV、机器人交易等场景。

## 1. 核心功能
- 余额查询：可以查询钱包BNB和BEP-20代币余额
- 定制交易：可以发送定制的普通转账和合约调用交易
- Swap支持：可以进行标准的v2和v3的Swap交易 
- 策略发送：支持直接发送交易、触发发送交易、循环发送交易
- 贿赂交易：可以进行贿赂交易，加快交易上链
